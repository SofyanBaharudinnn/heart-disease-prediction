{% if current_lang == 'en' %}About the HeartGuard Project{% else %}Tentang Proyek HeartGuard{% endif %}

{% if current_lang == 'en' %} Artificial intelligence-based heart disease prediction digital platform, designed to support rapid, accurate, and reliable initial clinical diagnosis. {% else %} Platform digital prediksi penyakit jantung berbasis kecerdasan buatan, dirancang untuk mendukung diagnosis klinis awal secara cepat, akurat, dan tepercaya. {% endif %}

{% if current_lang == 'en' %}Project Core Pillars{% else %}Pilar Utama Proyek{% endif %}

Random Forest

{% if current_lang == 'en' %} Machine Learning ensemble algorithm that builds hundreds of decision trees in parallel to predict heart disease risk probability robustly. {% else %} Algoritma Machine Learning ensemble yang membangun ratusan decision tree secara paralel untuk memprediksi tingkat probabilitas risiko penyakit jantung secara robust. {% endif %}

Multi-Holdout

{% if current_lang == 'en' %} Iterative cross-validation split method to evaluate model performance stability and ensure consistent accuracy estimates. {% else %} Metode evaluasi silang bertahap yang membagi dataset berkali-kali untuk melatih model secara iteratif guna menghindari bias dan memastikan estimasi akurasi yang konsisten. {% endif %}

UCI Dataset

{% if current_lang == 'en' %} Uses the internationally standard clinical dataset from the UCI Machine Learning Repository, covering 13 vital health parameters. {% else %} Menggunakan dataset klinis standar internasional dari UCI Machine Learning Repository yang mencakup 13 parameter kesehatan vital untuk evaluasi menyeluruh. {% endif %}

{% if current_lang == 'en' %}System Workflow Timeline{% else %}Alur Aliran Sistem{% endif %}
1

{% if current_lang == 'en' %}Medical Dataset Preparation{% else %}Persiapan Dataset Medis{% endif %}

{% if current_lang == 'en' %} Load dataset file heart.csv locally, detect and handle missing values, duplicates, and format alignment. {% else %} Memuat file dataset heart.csv secara lokal, mendeteksi dan menangani missing values, duplikasi data, serta melakukan penyelarasan format. {% endif %}

2

{% if current_lang == 'en' %}Multi-Holdout Validation{% else %}Validasi Multi-Holdout{% endif %}

{% if current_lang == 'en' %} Dataset is split into training (70%) and testing (30%) portions randomly for N iterations to evaluate prediction consistency. {% else %} Dataset dibagi menjadi porsi pelatihan (70%) dan pengujian (30%) secara acak sebanyak N iterasi untuk mengevaluasi konsistensi prediksi model secara acak. {% endif %}

3

{% if current_lang == 'en' %}Model Performance Evaluation{% else %}Evaluasi Kinerja Model{% endif %}

{% if current_lang == 'en' %} Calculate average evaluation metrics (Accuracy, Precision, Recall, F1-Score, and ROC-AUC) to assess aggregate model reliability. {% else %} Menghitung rata-rata metrik evaluasi model (Akurasi, Presisi, Sensitivitas, F1-Score, dan ROC-AUC) untuk menilai kehandalan performa model secara agregat. {% endif %}

4

{% if current_lang == 'en' %}Best Model Persistence{% else %}Persistensi Model Terbaik{% endif %}

{% if current_lang == 'en' %} Random Forest model with optimal F1-Score is saved as a serialized file (.pkl) using joblib for real-time predictions. {% else %} Model Random Forest dengan F1-Score optimal disimpan ke dalam format serialized file (.pkl) menggunakan modul joblib agar siap digunakan secara real-time. {% endif %}

5

{% if current_lang == 'en' %}Real-Time Prediction Analysis{% else %}Analisis Prediksi Real-Time{% endif %}

{% if current_lang == 'en' %} Accept clinical parameters from user web inputs, normalize data, and display heart disease probability and risk levels. {% else %} Menerima parameter klinis yang diinputkan pengguna melalui halaman web, menormalkan data, dan menyajikan probabilitas penyakit jantung beserta tingkat risikonya. {% endif %}
