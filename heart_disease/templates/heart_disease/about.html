Sistem Pakar Medis

Tentang Proyek HeartGuard

Platform digital prediksi penyakit jantung berbasis kecerdasan buatan, dirancang untuk mendukung diagnosis klinis awal secara cepat, akurat, dan tepercaya.

Pilar Utama Proyek

Random Forest

Algoritma Machine Learning ensemble yang membangun ratusan decision tree secara paralel untuk memprediksi tingkat probabilitas risiko penyakit jantung secara robust.

Multi-Holdout

Metode evaluasi silang bertahap yang membagi dataset berkali-kali untuk melatih model secara iteratif guna menghindari bias dan memastikan estimasi akurasi yang konsisten.

UCI Dataset

Menggunakan dataset klinis standar internasional dari UCI Machine Learning Repository yang mencakup 13 parameter kesehatan vital untuk evaluasi menyeluruh.

Teknologi yang Digunakan
Django 4.2
Web Application Framework berbasis Python yang aman dan skalabel.
Scikit-learn
Pustaka Python utama untuk pelatihan dan evaluasi model Machine Learning.
Pandas & NumPy
Manipulasi struktur data berkinerja tinggi serta komputasi matriks numerik.
Matplotlib & Seaborn
Pembuat grafik visualisasi analisis korelasi data medis dan metrik model.
Bootstrap 5 & Icons
Framework frontend responsive untuk tampilan tata letak antarmuka web modern.
SQLite (Django ORM)
Penyimpanan relasional lokal untuk manajemen data riwayat medis pasien.
Alur Aliran Sistem
1

Persiapan Dataset Medis

Memuat file dataset heart.csv secara lokal, mendeteksi dan menangani missing values, duplikasi data, serta melakukan penyelarasan format.

2

Validasi Multi-Holdout

Dataset dibagi menjadi porsi pelatihan (80%) dan pengujian (20%) secara acak sebanyak N iterasi untuk mengevaluasi konsistensi prediksi model secara acak.

3

Evaluasi Kinerja Model

Menghitung rata-rata metrik evaluasi model (Akurasi, Presisi, Sensitivitas, F1-Score, dan ROC-AUC) untuk menilai kehandalan performa model secara agregat.

4

Persistensi Model Terbaik

Model Random Forest dengan F1-Score optimal disimpan ke dalam format serialized file (.pkl) menggunakan modul joblib agar siap digunakan secara real-time.

5

Analisis Prediksi Real-Time

Menerima parameter klinis yang diinputkan pengguna melalui halaman web, menormalkan data, dan menyajikan probabilitas penyakit jantung beserta tingkat risikonya.
